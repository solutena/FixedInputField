# FixedInputField

한글 입력 버그를 해결한 인풋 필드입니다.

## 기존 문제점

1. 한글을 입력 할 때 뒷 글자가 한글인 경우 한칸 건너뛰면서 입력되는 현상
2. 범위를 선택한 채로 한글 입력시 범위가 지워지지 않는 현상
3. 글자 제한을 넣은 상태로 한글 입력시 오류가 뜨는 현상
